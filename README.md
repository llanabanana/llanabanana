## Projects

### [mrs-taxi-app](https://github.com/llanabanana/mrs-taxi-app)
Full-stack ride-sharing platform developed in a team of 3 across five concurrent university courses (Server-side, Client-side, Mobile Apps, Software Testing, Methodologies). Spring Boot REST API, Angular web app, and native Android app with real-time tracking, driver matching, notifications, and ride management. Two semesters of Scrum sprints, pull requests, code reviews, and CI. This was the most comprehensive project of my studies -- it taught me how real teams ship software.

### [kv-engine-go](https://github.com/llanabanana/kv-engine-go)
A key-value storage engine built in Go from scratch. Implements LSM-tree architecture with write-ahead log, memtable, SSTable with multi-level compaction, bloom filter, LRU cache, token bucket rate limiter, SimHash, and Count-Min Sketch. Built for the Advanced Algorithms and Data Structures course.

### [webscraper-cpp](https://github.com/llanabanana/webscraper-cpp)
A concurrent web scraper in C++ using Intel TBB for parallelism and libcurl for HTTP. Custom HTML parser with regex-based extraction. Scrapes, parses, and analyzes book data across paginated pages.

### [sok-projekat](https://github.com/llanabanana/sok-projekat)
Plugin-based graph visualization platform built with Django. Uses Python entry points for plugin discovery, supporting multiple data source formats and visualization strategies. Includes a web interface for interactive graph exploration.
